## How to Run

### 1. Clone the repository
<pre>
git clone https://github.com/your-username/todoapp.git

</pre>
### 2. Install dependencies
Backend:
<pre>
npm install
</pre>
Frontend:
<pre>
npm install
</pre>
### 3. Run the backend
<pre>
npm start
Backend runs on: http://localhost:3001
</pre>

### 4. Run the frontend
<pre>
ng serve
Frontend available on: 
http://localhost:4200 </pre>
